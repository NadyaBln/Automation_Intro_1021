##How to install:
	Open solution with Visual Studio;
	For solution you need NuGet packages:
		- Selenium.WebDriver;
		- NUnit;
		- Microsoft.NET.Test.Sdk;
		- Nunit3TestAdapter;	

	Please put folder 'Resources' into 'Project Folder' -> \bin\Debug\net5.0
